Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
Выведет числа от 0 до 9, затем будет deadlock, т.к. цикл с range по каналу
остановится, если канал будет закрыт, а в данном случае канал не закрывают.
